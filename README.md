# swiflearn-assignment

# Technologies Used
1. React Js
2. React dom
3. Next Js
4. Typescript

# Steps to Follow
1. Clone the repo.
2. Open terminal.
3. Enter into the directory like:- cd /var/www/repo_name
4. Run command "npm install"
5. Run command "npm run dev"
6. Once bundled updated open url "http://localhost:3030"
